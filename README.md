# Smart Study Planner

A Python-based Smart Study Planner that helps students create efficient and balanced study schedules based on subject difficulty and study time requirements.

## 📌 Overview

Smart Study Planner is designed to assist students in organizing their study sessions effectively. By taking subject difficulty levels and available study time into account, the planner generates a personalized timetable that promotes balanced learning and better time management.

## ✨ Features

* Add multiple subjects
* Assign difficulty levels to each subject
* Specify study hours for every subject
* Generate a balanced study timetable
* Prioritize difficult subjects appropriately
* Easy-to-use command-line interface
* Helps improve productivity and study efficiency

## 🛠️ Technologies Used

* Python 3
* Built-in Python Libraries

  * random
  * copy
  * math

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/siddhardha-roy/smart-study-planner.git
```

2. Navigate to the project folder:

```bash
cd smart-study-planner
```

3. Run the program:

```bash
python main.py
```

## 📖 Usage

1. Enter the subjects you want to study.
2. Specify the difficulty level for each subject.
3. Enter the required study time.
4. Generate the study plan.
5. Follow the generated timetable for effective learning.
